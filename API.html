多数据源融合的互联网个人贷款及全流程风控系统

1.文档信息

  • 项目名称：多数据源融合的互联网个人贷款及全流程风控系统。
  • 版本：0.0.1
  • 描述：本文档定义了个人贷款系统前后端交互的接口规范，所有接口遵循RESTful设计原则。

2.通用约定

基础信息

  • Base URL：https://后端服务器IP
  • 数据格式：请求与响应主体均使用 application/json。
  • 字符编码：UTF-8

返回码

  • 若请求成功，API会返回如下JSON格式的返回码及说明
{
    "code": "***",
    "data": {
        ***
    }
}
  • 若请求失败，API会返回如下JSON格式的返回码及说明
{
    "error":{
        "code": "***",
        "message": "*****"
    }
}

token鉴权

  • Authorization: Bearer

标准HTTP状态码

状态码 含义 说明
200 OK 请求成功
201 Created 资源创建成功
400 Bad Request 请求参数错误
401 Unauthorized 未认证或Token无效
403 Forbidden 权限不足
404 Not Found 请求的资源不存在
500 Internal Server Error 服务器内部错误

3.接口详述

3.1 用户注册登录

3.1.1 用户注册

  • 功能说明：用户输入用户名、密码和角色注册账号 ，密码要求有不少于8位，包含字母、数字、特殊符号三类，且必须包含大小写，并且输入两次进行检查，角色为“管理员”和“普通用户”二选一。
  • 功能需求：FR-01
  • Endpoint: POST https://后端服务器IP/api/register (Basse URL以下皆省略，即省略“https://后端服务器IP/api”)
  • 请求体：
{
    "userName": "sjm",
    "password": "sjm123456",
    "role": "普通用户"
}
  • 请求参数
关键字 类型 含义
userName String 用户名
password String 密码
role String 角色
  • 成功响应
{
    "code": 201,
    "data": {
        "userId": 1,
        "userName": "sjm",
        "password": "sjm123456",
        "role": "普通用户"
    },
    "error": null
}
  • 响应参数
关键字 类型 含义
code int 返回码
userId long 用户账号
userName String 用户名
password String 密码
role String 角色

3.1.2 用户登录

  • 功能说明：用户使用用户名和密码进行登录。
  • 功能需求：FR-01
  • Endpoint: POST /login
  • 请求体
{
    "userName": "sjm",
    "password": "sjm123456"
}
  • 请求参数
关键字 类型 含义
userName String 用户名
password String 密码
  • 成功响应
{
    "code": 200,
    "data": {
        "userId": 1,
        "userName": "sjm",
        "role": "普通用户",
        "token": "************"
    },
    "error": null
}
  • 响应参数
关键字 类型 含义
code int 返回码
userId long 用户账号
userName String 用户名
role String 角色

3.2 用户完善身份信息，管理员管理用户信息

3.2.1 用户完善身份信息

  • 功能说明：用户填写真实姓名、性别、年龄和身份证号，身份证号为18位。
  • 功能需求：FR-02
  • Endpoint:POST /user/userInfo
  • Authorization: Bearer
  • 请求体
{
    "userId": 1,
    "realName": "玄小盾",
    "sex": "男",
    "age": 18,
    "idNumber": "110000200601010001"
}
  • 请求参数
关键字 类型 含义
realName String 用户真实姓名
sex String 用户性别
age int 用户年龄
idNumber String 用户身份证号
  • 成功响应
{
    "code": 201,
    "data": {
        "userId": 1,
        "userName": "sjm",
        "realName": "玄小盾",
        "sex": "男",
        "age": 18,
        "idNumber": "110000200601010001"
    },
    "error": null
    }
  • 响应参数
关键字 类型 含义
code int 响应码
userId long 用户账号
userName String 用户名
realName String 用户真实姓名
sex String 用户性别
age int 用户年龄
idNumber String 用户身份证号

    "page": 1,
    "size": 10,
    "users": [

        "userId": 1,
        "userName": "sjm",
        "password": "sjm123456",
        "realName": "玄小盾",
        "sex": "男",
        "age": 18,
        "idNumber": "110000200601010001",
        "cardNumber": null
      },
      {
        "userId": 2,
        "userName": "fmy",
        "password": "xdd123456",
        "realName": "玄大盾",
        "sex": "男",
        "age": 18,
        "idNumber": "110000200602020002",
        "cardNumber": null
      }
    ]
  },
  "error": null
}

关键字 类型 含义
code int 响应码
total int 总记录数

3.2.3 管理员强制注销风险账户

  • 功能说明：管理员通过账号删除用户，仅可删除普通用户，前端请将这个功能集成到3.2.2总览中。

3.3 贷款

3.3.1 获取贷款产品列表

  • 功能说明：用户申请贷款前，需查看可申请的贷款产品。管理员同样借此查看产品总览，前端注意管理员多了删除键
  • 功能需求：FR-03
  • Endpoint: GET /loan/productInfo?page={page}&size={size}
  • 成功响应
{
  "code": 200,
  "data": {
    "total": 100,
    "product":[
      {
        "loanId": 1,
        "loanName": "王八贷",
        "loanTermMax": 10,    
        "interestRate": 0.12,
        "loanStatus": "已开放"
      }
      // 其他产品
    ]
  },
  "error": null
}
  • 响应参数
关键字 类型 含义
code int 响应码
total int 总记录数
page int 当前记录页
size int 每页条数
product list 贷款产品列表，包含loanId,loanName,loanTermMax,interestRate,LoanStatus

3.3.2 用户提交贷款申请

  • 功能说明：用户选择产品，填写金额、期限、还款方式，提交贷款申请,状态：未申请-审核中-已通过/未通过~已放款/未通过
  • 功能需求：FR-03
  • Endpoint: POST /loan/user/application
  • Authorization: Bearer
  • 请求体
{ 
  "userId": 1,
  "loanId": 1,
  "userLoanAmount": 10000.00,
  "userLoanTerm": 6,
  "userLoanWay": "等额本息"
}
  • 请求参数
关键字 类型 含义
userId long 申请人账号
loanId long 贷款产品号
userLoanAmount double 用户贷款金额
userLoanTerm int 用户贷款分期数
userLoanWay String 用户还款方式
  • 成功响应
{
  "code": 201,
  "data": {
    "userLoanStatus": "审核中"
  },
  "error": null
 
}
  • 响应参数
关键字 类型 含义
userLoanStatus String 贷款申请状态
